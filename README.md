# Plataforma Web de Análisis de Datos

Proyecto desarrollado como parte del Proyecto Productivo del SENA.

## Descripción

Aplicación web para la carga, procesamiento y análisis de conjuntos de datos.

## Tecnologías utilizadas

- HTML
- CSS
- JavaScript
- Node.js
- Bases de datos SQL

## Funcionalidades actuales

- Carga de archivos CSV y Excel
- Limpieza de datos
- Exploración de información
- Estadísticas descriptivas
- Gestión de usuarios
- Dashboard interactivo

## Estado del proyecto

En desarrollo.
